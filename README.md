# Coronary-Artery-Disease-Identification-from-CT-Scan-Images-using-Machine-Learning
Coronary artery disease, or CAD, is one of the leading causes of death worldwide. Early and accurate detection can save lives, but traditional diagnosis often requires invasive and time-consuming procedures. With the advancement of CT scan imaging and machine learning, it is now possible to detect CAD more quickly and efficiently.
